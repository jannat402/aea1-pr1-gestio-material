# aea1-pr1-gestio-material
Pràctica 1 – Sistema de gestió de material
Descripció
Aquest projecte és una primera versió d’un sistema de gestió de material per a l’Institut Camí de Mar. Permet registrar materials, organitzar-los per categories, gestionar préstecs i devolucions, i consultar la disponibilitat.

Estructura del repositori
src/ → Codi font Java (paquet prt1).
doc → Carpeta amb el Javadoc generat.
Documentació d’usuari en format PDF.
Documnetació tècnica en format PDF.
README.md → Guia ràpida del projecte.

Requisits
Java 17 o superior.
Eclipse IDE (o qualsevol entorn Java).
Connexió a Internet per descarregar el projecte.

Guia d’ús bàsic
1. Afegir material → introdueix ID, nom, descripció, categoria i quantitat.
2. Fer préstec → introdueix ID del préstec, ID del material, prestatari, quantitat i dies.
3. Retornar préstec → introdueix l’ID del préstec.
4. Buscar material per ID → mostra les dades d’un material concret.
5. Llistar materials → mostra tots els materials.
